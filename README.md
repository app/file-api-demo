Пример вставки в страницу файла с локального диска через Upload.  
То есть файл сразу после аплоада, без передачи на сервер используется на странице.

Загрузив текстовый файл, видим его содержимое.
Пример — в text.html

Загрузив файл с картинкой, получаем картинку на странице.
Пример — в image.html

Загрузив js файл, запустим на исполнение содержащийся в нем код.
Пример — в script.html


Используется File API поддерживаемое всеми основными современными браузерами.  
Пример взят и доработан с сайта  
http://blog.teamtreehouse.com/reading-files-using-the-html5-filereader-api
